Help and Support

There is a wealth of community created content, blogs and tutorials available.

If you want to find an answer to a more specific problem, you can ask it on our GitHub Discussions page.

You can also chat to other MonoGame users via Discord.

Bugs and feature requests

If you find a bug or have a feature request, please open a new issue. Before opening any issue, please search for existing issues.

Please note that the issue tracker is not for requesting help.
